# Aryan Virani - Personal Portfolio

A modern, high-performance personal portfolio built with React, TypeScript, and Vite. Designed to showcase my projects, skills, and professional experience with a focus on smooth animations, clean aesthetics, and excellent user experience.

## ✨ Features

- **Modern Tech Stack**: React 19, TypeScript, Vite, Tailwind CSS v4.
- **Smooth Animations**: Powered by Framer Motion for scroll reveals, hover effects, custom cursors, and ambient backgrounds.
- **Responsive Design**: Fully responsive layout that looks perfect on mobile, tablet, and desktop devices.
- **Dark/Light Mode**: Integrated theme toggling for personalized viewing.
- **Custom UI Components**: Built using Shadcn UI (Radix primitives) for accessible and highly customizable components.
- **Performance Optimized**: Smooth scrolling, optimized rendering, and fast loading speeds.

## 🛠️ Technologies Used

- **Framework/Library**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Routing**: [TanStack Router](https://tanstack.com/router/latest)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Components**: [Radix UI](https://www.radix-ui.com/) / shadcn/ui

## 📂 Project Structure

- `src/components/`: Reusable UI components (Cursor, Preloader, Cards, Reveal, Ambient Background, etc.).
- `src/routes/`: Page components and routing logic (powered by TanStack Router).
- `src/hooks/`: Custom React hooks.
- `src/lib/`: Utility functions and configuration.
- `public/`: Static assets like images and resumes.

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository (if applicable) or navigate to the project directory:
   ```bash
   git clone https://github.com/AryanPatel03/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` to view the app.

### Building for Production

To build the application for production, run:
```bash
npm run build
```
The optimized files will be generated in the `dist` or `build` folder.
